Documentation

How to use ImberLab

Tap any module to expand the instructions.

🧮

Calculations

Factor, Gravimetric & Volumetric

▾

Three calculation types for analytical chemistry work:

  1. Open Calculations from the main menu and choose Factor, Gravimetric or Volumetric.
  2. Enter your values in the input fields. Decimal input is supported.
  3. Tap Calculate to get the result instantly.
  4. If Save Results is enabled in Settings, the result is automatically saved to Analysis Results.
💡 Use the Formula Builder to create your own custom formulas beyond the built-in types.
📊

ImbaControl

Statistical Quality Control

▾

Four statistical tools for process monitoring and QC:

  1. Control Chart — Enter measured values to visualize process stability with mean, UCL and LCL lines.
  2. Outlier Test — Detects outliers in your dataset using statistical criteria.
  3. Stability Trend — Analyzes whether your process is trending up, down or stable.
  4. Calibration Curve — Input standard values to generate a linear calibration curve with R² value.
💡 Results from ImbaControl are auto-saved to Analysis Results if saving is enabled.
🔣

Formula Builder

Custom calculation formulas

▾

Build and save your own formulas using letters as variables:

  1. Enter a name for your formula (e.g. "Yield %").
  2. Type your formula using letters as variables and operators: + - * / ^ ( )
  3. Enter the result unit (e.g. g/l, %, mmol).
  4. Tap Detect Variables — ImberLab automatically finds all variables in your formula.
  5. Enter values for each variable and tap Calculate.
  6. Tap Save Formula to store it for reuse. (Premium: unlimited saves)
💡 Example: ((V*MG)/(e*d*v*1000))*5 — use any letter combinations as variable names.
📋

Logbook

Device calibration & maintenance records

▾

Track all service events for your lab devices:

  1. Tap + Entry to create a new entry. Select the device, type, date and optional next due date.
  2. Each device gets its own sequential entry numbering (#1, #2, …).
  3. Due dates are color-coded: green = OK, orange = due soon, red = overdue.
  4. Tap Photo (Premium) to attach a photo from camera or gallery.
  5. Tap Device to jump to the device profile. Tap Export to share via email.
  6. Long-press a card to delete all entries for that device.
💡 Search by device name, date, person or note text using the search bar.
⚗️

Chemicals & GHS

Chemical inventory management

▾

250+ pre-loaded chemicals with full GHS data:

  1. Browse or search chemicals by name or CAS number.
  2. Tap a chemical to see GHS pictograms, H/P statements, storage notes and PPE requirements.
  3. Tap + Add to create a custom chemical entry.
  4. Long-press a custom chemical to edit or delete it.
  5. Tap the camera icon (Premium) to attach a photo.
  6. Tap the ? button for detailed GHS hazard information.
💡 Custom chemicals are included in Backup/Export.
🖥️

Devices & QR

Lab instrument management

▾

Manage your complete lab device inventory:

  1. Browse devices by category or search across all fields.
  2. Tap a device to open its profile: type, serial number, manufacturer, location, description.
  3. Each device has a QR code generated from its serial number.
  4. Tap the edit icon to update location or description — changes are auto-logged in the Logbook.
  5. Use Decommission to retire a device from all lists while preserving its history.
  6. Tap Reactivate on a decommissioned device's logbook entry to restore it.
📈

Analysis Results

Auto-saved calculation history

▾

All calculation results are stored here automatically:

  1. Enable Save Results in Settings to start auto-saving.
  2. Tap a result card to expand and see full details.
  3. Tap Photo (Premium) to attach a document photo.
  4. Tap Note to add a free-text comment.
  5. Tap Scan to scan a label via OCR and auto-fill the label field.
  6. Long-press a card to delete that result.
💡 Settings → Delete All Results clears the entire history at once.
🔑

Password Manager

Secure credential storage

▾

Store and manage lab credentials — all data stays on your device:

  1. Tap + Add Entry to create a credential with title, username, password, URL and notes.
  2. Tap the star to mark an entry as favorite — favorites appear at the top.
  3. Tap an entry to expand it. Use the eye icon to reveal the password and history together.
  4. Password History is tracked automatically when you edit a password.
  5. Tap Generator to create a strong random password.
  6. Long-press an entry to edit or delete it.
💡 Free: up to 3 entries. Premium: unlimited entries.
⚙️

Settings

App configuration & backup

▾
  1. Save Results — toggle auto-saving of all calculation results.
  2. App Lock — protect the app with a PIN or fingerprint.
  3. Backup — export all app data to a local file and restore it anytime.
  4. Premium — purchase or view your Premium status.
  5. Delete All Results / Logbook — permanently clear history data.
  6. Rate App — opens the Play Store review page.
💡 Always create a Backup before uninstalling or switching devices.
